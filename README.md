# Personal-Finance-Tracker

💰 Personal Finance Tracker (Python + PySide6 + MySQL)

A desktop Personal Finance Tracker built with Python, PySide6, and MySQL.
This application helps users manage their daily finances by tracking income, expenses, and current balance in a simple graphical interface.
The system automatically calculates the balance and prevents users from adding expenses that would make the balance negative.

✨ Features

💰 Add Money
Record income with amount, category, note, and timestamp.
Categories include salary, gifts, parent money, etc.

💸 Expense Management
Track spending with categories like food, transport, gadgets, etc.
Prevents expenses if the balance is insufficient.

🧾 Transaction History
View all income records.
View all expense records.
Data displayed in organized tables.

📊 Automatic Balance Calculation
Balance = Total Income − Total Expenses
Updates instantly after every transaction.

🗄 MySQL Database Integration
Uses MySQL to store all financial data.
Automatically creates database and tables if they do not exist.

🖥 Modern GUI
Built using PySide6 (Qt for Python).
Tab-based interface.
Clean and user-friendly design.

🛠 Technologies Used
Python
PySide6 (Qt GUI Framework)
MySQL
mysql-connector-python
